Thank you for taking an interest in my game!

Immortal blade is a turn based RPG with a strong emphasis on the game's main characters, both narratively and in gameplay. Some of my primary inspirations for this game include 'Xenoblade Chronicles,' 'Paper Mario,' and 'Steven Universe.'

The game (and this website) are still a work in progress, so bear with me as I continue to make progress and add new content.
